# Bank_Power_BI
Bank Dashboard — ушбу лойиҳада банк операциялари бўйича кирим ва чиқимларни таҳлил қилинган. Дашборд орқали маълумотларни йиллар, ойлар ва валюталар (USD, EUR, UZS) кесимида кўриш мумкин. KPI карталар, инфлов–аутфлов таққослаши ва (YoY) ўсиш кўрсаткичлари мавжуд. Лойиҳа молиявий назорат, таҳлил ва бошқарув қарорларини қабул қилишда қўл келади.
